# Project6 - Blockchain Supply Chain Project

## Installation

#### Install project dependencies:

```
$ npm install -g truffle@4.1.15
$ npm install --save truffle-hdwallet-provider@1.0.17

npm install webpack-dev-server -g
npm install web3

$ npm build
```
